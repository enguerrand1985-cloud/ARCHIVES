# SUPPLIER__SONGWAYPET.narrative

## Objet

Ce fichier restitue le narratif fournisseur SongwayPet / Hebei Yuanfeng pour expliquer le cycle complet du premier RUN Meow Paw Paw : origine de la relation, construction produit, préparation Amazon FBA, échec du rail DDP, remboursement Alibaba et annulation opérationnelle.

Il complète les fichiers structurés Source of Truth, sans les remplacer.

## Résumé exécutif

SongwayPet / Hebei Yuanfeng a été le premier fournisseur RUN pressenti pour Meow Paw Paw. La relation s’est construite progressivement à partir de discussions Alibaba, samples, tests packaging, logo/stickers, et faisabilité d’un lancement Amazon FBA France.

Le fournisseur avait donné des signaux favorables sur le DDP France et avait accompagné la préparation du run jusqu’à un stade très avancé : packaging, FNSKU, cartonisation, plan FBA, labels cartons Amazon et paiement du solde.

Le run a finalement été annulé avant expédition car le forwarder fournisseur n’a pas accepté de gérer le DDP / customs clearance pour le produit contenant du catnip / matière végétale. La bascule vers un modèle DAP / dédouanement acheteur n’a pas été acceptée par TRADE-VISION. Le remboursement Alibaba a été demandé, accepté et visible côté Qonto. L’expédition FBA a été annulée dans Seller Central.

Meow Paw Paw reste actif. Le fournisseur SongwayPet est suspendu pour RUN tant qu’un forwarder ne confirme pas explicitement le DDP du produit exact avant paiement.

## 1. Origine de la relation

La relation avec SongwayPet a commencé par des échanges Alibaba autour de produits naturels pour chats, de samples, de packaging, de catnip/silvervine/gallfruit et de possibilités DDP France.

Le fournisseur a progressivement été considéré comme fournisseur potentiel car il répondait sur :
- les samples ;
- le packaging ;
- la personnalisation ;
- les stickers / logo ;
- les dimensions sachets ;
- le transport DDP France ;
- la préparation Amazon FBA.

Ce point est important : la commande finale n’a pas été faite sans historique. Le fournisseur avait été travaillé sur plusieurs mois.

## 2. Construction produit et packaging

Le positionnement Meow Paw Paw visait un produit naturel, premium, kawaii et proprement présenté. Les échanges avec SongwayPet ont couvert :
- vente unitaire premium ;
- sachets OPP ;
- alternatives kraft / Dolly Bag ;
- stickers transparents ;
- inserts ;
- logo Meow Paw Paw ;
- back labels ;
- warning sac plastique ;
- FNSKU Amazon ;
- packaging compatible FBA.

Les choix initiaux ont mené à deux SKU RUN :
- MPP-BAT-STD-MAT-V1 ;
- MPP-BUN-STD-1BOU-S-V1.

## 3. Passage au RUN Amazon FBA

Le run final était un petit lancement test de 600 unités :
- 300 unités MPP-BAT-STD-MAT-V1 ;
- 300 unités MPP-BUN-STD-1BOU-S-V1.

Le rail convenu était DDP / transport aérien / préparation Amazon FBA, avec application des stickers, FNSKU et labels cartons Amazon côté fournisseur.

Le run a atteint un stade avancé :
- packaging finalisé ;
- FNSKU transmis ;
- photos / QC reçus ;
- cartonisation reçue ;
- plan FBA créé ;
- labels cartons générés ;
- labels cartons transmis au fournisseur ;
- solde payé.

## 4. Dossier conformité et lecture import

Le dossier import/compliance avait été travaillé par couches :
- confirmations fournisseur ;
- documents contractuels ;
- rapports techniques ;
- preuves matières ;
- lecture réglementaire externe ;
- échanges avec les autorités françaises.

Cette lecture réduisait certains risques, mais n’a pas suffi à sécuriser l’exécution DDP opérationnelle par le forwarder.

Point clé retenu : une lecture réglementaire ou une confirmation fournisseur n’équivaut pas à une confirmation forwarder DDP pour le produit exact.

## 5. Incident DDP / customs clearance

Après paiement du solde, le fournisseur a indiqué que son agent / forwarder ne pouvait pas assurer le DDP / customs clearance pour le produit, car le catnip était considéré comme une plante / matière végétale.

Vicky / Vicci Chen a indiqué avoir sollicité plusieurs freight forwarders, qui auraient confirmé le même blocage. Le transport pouvait éventuellement être fait, mais le dédouanement devait être porté côté client.

Ce scénario revenait à basculer le rail DDP prévu vers un rail DAP / dédouanement acheteur, non accepté par TRADE-VISION.

## 6. Décision opérationnelle

TRADE-VISION a refusé :
- d’accepter une expédition sans DDP réel ;
- d’assumer un dédouanement improvisé ;
- de basculer en DAP sans transitaire identifié, devis, responsabilité douane claire et recalcul de marge.

La décision a été :
- ne pas expédier ;
- demander remboursement ;
- annuler le plan FBA ;
- geler le lancement commercial ;
- conserver les actifs Amazon dormants.

## 7. Remboursement Alibaba / Qonto

Alibaba / Trade Assurance a accepté le remboursement du run. Le remboursement bancaire est visible côté Qonto. Le dossier reste à rapprocher définitivement avec le relevé Qonto de mai pour classer l’écart résiduel.

Lecture actuelle :
- remboursement quasi complet ;
- écart net provisoire lié probablement à frais / change / traitement bancaire ;
- pas de perte fournisseur définitive majeure confirmée à ce stade.

## 8. Annulation Amazon FBA

Le plan FBA Amazon FBA15LSK0YD6 avait été créé, puis annulé après l’échec DDP.

Conséquences :
- aucun stock expédié ;
- aucun tracking à saisir ;
- unités attendues : 0 ;
- unités localisées : 0 ;
- labels cartons invalidés ;
- listings / ASIN / SKU / FNSKU conservés dormants ;
- aucun PPC, Vine, coupon ou lancement commercial.

## 9. Statut fournisseur après incident

SongwayPet / Hebei Yuanfeng n’est plus fournisseur RUN éligible en l’état.

Statut :
- suspended_ddp_customs_failure ;
- blocked_not_run_eligible_without_forwarder_confirmed_ddp.

Condition de réactivation :
obtenir avant tout paiement une confirmation écrite du forwarder couvrant explicitement :
- le produit exact ;
- catnip / silvervine / gallfruit / matière végétale naturelle ;
- DDP door-to-door Amazon FBA France ;
- customs clearance handled by supplier/forwarder ;
- no buyer customs clearance required.

## 10. Leçon principale

Supplier-level DDP claim is not enough.

Pour tout futur produit naturel contenant catnip, silvervine, gallfruit ou autre matière végétale sensible, la confirmation DDP doit venir du forwarder ou être explicitement validée par lui avant paiement.
